## 4. 진행 방식

1. **스터디 이틀 전까지** 궁금한 사항, 더 이야기하고 싶은 사항을 신청
2. **스터디 하루 전부터** 각 질문에 대해 자료조사 및 답변 준비
3. **스터디 날에** 각 질문에 대해 토의
